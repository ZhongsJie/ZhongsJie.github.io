NginxжҢҮеҚ—

Nginx д»Ӣз»Қ#

вҖңNginx жҳҜдёҖж¬ҫиҪ»йҮҸзә§зҡ„ HTTP жңҚеҠЎеҷЁпјҢйҮҮз”ЁдәӢд»¶й©ұеҠЁзҡ„ејӮжӯҘйқһйҳ»еЎһеӨ„зҗҶж–№ејҸжЎҶжһ¶пјҢиҝҷи®©е…¶е…·жңүжһҒеҘҪзҡ„ IO жҖ§иғҪпјҢж—¶еёёз”ЁдәҺжңҚеҠЎз«Ҝзҡ„еҸҚеҗ‘д»ЈзҗҶе’ҢиҙҹиҪҪеқҮиЎЎгҖӮвҖқ

  1. жӯЈеҗ‘д»ЈзҗҶ
    • жӯЈеҗ‘д»ЈзҗҶвҖңд»ЈзҗҶвҖқзҡ„жҳҜе®ўжҲ·з«ҜпјҢиҖҢдё”е®ўжҲ·з«ҜжҳҜзҹҘйҒ“зӣ®ж Үзҡ„пјҢиҖҢзӣ®ж ҮжҳҜдёҚзҹҘйҒ“е®ўжҲ·з«ҜжҳҜйҖҡиҝҮVPNи®ҝй—®зҡ„гҖӮ
  2. еҸҚеҗ‘д»ЈзҗҶ
    • еҸҚеҗ‘д»ЈзҗҶвҖңд»ЈзҗҶвҖқзҡ„жҳҜжңҚеҠЎеҷЁз«ҜпјҢиҖҢдё”иҝҷдёҖдёӘиҝҮзЁӢеҜ№дәҺе®ўжҲ·з«ҜиҖҢиЁҖжҳҜйҖҸжҳҺзҡ„гҖӮ

Nginxзҡ„Master-WorkerжЁЎејҸ#

  1. Master
    • иҜ»еҸ–е№¶й…ҚзҪ®nginx.conf, з®ЎзҗҶWorkerиҝӣзЁӢ
  2. Worker
    • з»ҙжҠӨдёҖдёӘзәҝзЁӢпјҲйҒҝе…ҚзәҝзЁӢеҲҮжҚўпјүпјҢеӨ„зҗҶиҝһжҺҘе’ҢиҜ·жұӮпјӣжіЁж„ҸWorkerиҝӣзЁӢзҡ„дёӘж•°з”ұй…ҚзҪ®ж–Үд»¶еҶіе®ҡпјҢдёҖиҲ¬е’ҢCPUдёӘж•°зӣёе…іпјҲжңүеҲ©дәҺиҝӣзЁӢеҲҮжҚўпјүпјҢй…ҚзҪ®еҮ дёӘе°ұжңүеҮ дёӘWorkerиҝӣзЁӢгҖӮ

NginxзғӯйғЁзҪІ#

  1. дҝ®ж”№й…ҚзҪ®ж–Үд»¶nginx.confеҗҺпјҢйҮҚж–°з”ҹжҲҗж–°зҡ„workerиҝӣзЁӢпјҢеҪ“з„¶дјҡд»Ҙж–°зҡ„й…ҚзҪ®иҝӣиЎҢеӨ„зҗҶиҜ·жұӮпјҢиҖҢдё”ж–°зҡ„иҜ·жұӮеҝ…йЎ»йғҪдәӨз»ҷж–°зҡ„workerиҝӣзЁӢпјҢиҮідәҺиҖҒзҡ„workerиҝӣзЁӢпјҢзӯүжҠҠйӮЈдәӣд»ҘеүҚзҡ„иҜ·жұӮеӨ„зҗҶе®ҢжҜ•еҗҺпјҢkillжҺү

Keepalived+Nginxе®һзҺ°й«ҳеҸҜз”Ё#

  1. KeepalivedжҳҜдёҖдёӘй«ҳеҸҜз”Ёи§ЈеҶіж–№жЎҲпјҢдё»иҰҒжҳҜз”ЁжқҘйҳІжӯўжңҚеҠЎеҷЁеҚ•зӮ№еҸ‘з”ҹж•…йҡң
  2. keepalivedжҳҜд»ҘVRRPеҚҸи®®дёәе®һзҺ°еҹәзЎҖзҡ„пјҢVRRPе…Ёз§°Virtual Router Redundancy ProtocolпјҢеҚіиҷҡжӢҹи·Ҝз”ұеҶ—дҪҷеҚҸи®®гҖӮ
  3. keepalivedдё»иҰҒжңүдёүдёӘжЁЎеқ—пјҢеҲҶеҲ«жҳҜcoreгҖҒcheckе’ҢvrrpгҖӮcoreжЁЎеқ—дёәkeepalivedзҡ„ж ёеҝғпјҢиҙҹиҙЈдё»иҝӣзЁӢзҡ„еҗҜеҠЁгҖҒз»ҙжҠӨд»ҘеҸҠе…ЁеұҖй…ҚзҪ®ж–Үд»¶зҡ„еҠ иҪҪе’Ңи§ЈжһҗгҖӮcheckиҙҹиҙЈеҒҘеә·жЈҖжҹҘпјҢеҢ…жӢ¬еёёи§Ғзҡ„еҗ„з§ҚжЈҖжҹҘж–№ејҸгҖӮvrrpжЁЎеқ—жҳҜжқҘе®һзҺ°VRRPеҚҸи®®зҡ„гҖӮ
